# demo

## 项目设置
```
npm install
```

### 启动
```
npm run serve
```

### 项目目录
```
|-- src
    |-- components
        |-- chart
            |-- options    // 存放各种图表的option
                |-- map    // 地图option
                    |-- index.js
            |-- index.vue    // echarts通用封装组件
    |-- App.vue // 图表使用
```
